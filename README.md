# OrderFlow

A full-featured online ordering system built with Django, designed to handle product browsing, user authentication, cart management, and order processing.

## Overview

OrderFlow is a web-based e-commerce application that enables customers to browse products, manage shopping carts, and complete orders through a secure authentication flow. The backend is built with Django and Django REST Framework, with a SQLite database for development.

## Features

- User registration and authentication with session management
- Product catalog with detailed views and image support
- Shopping cart functionality with quantity adjustment
- Order summary and checkout workflow
- Admin interface for product and order management
- Static file handling for media assets

## Technology Stack

- Backend: Python, Django, Django REST Framework
- Database: SQLite (development), configurable for PostgreSQL/MySQL
- Frontend: HTML5, CSS3, JavaScript
- Authentication: Django built-in auth system
- File Handling: Django staticfiles and media management
